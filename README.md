# Embeddings y Transformers

## Instalar dependencias

```bash
    npm install
```

## Crear base de datos

```bash
    npx prisma generate
    npx prisma db push
```

## Ejecutar el proyecto

```bash
    npm run dev
``` 

- Abrir localhost:3000 por que el frontend ya fue integrado
